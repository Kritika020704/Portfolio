# 🌟 Personal Portfolio – Kritika Gupta

A modern and responsive **personal portfolio website** showcasing my skills, projects, experience, and contact information.  
Built using **HTML, CSS, and JavaScript**, this portfolio highlights my journey as a **Frontend & Full-Stack Developer**.

---

## 🌐 Live Demo  
👉 **https://kritika020704.github.io/Portfolio/**

---

## ✨ Features  
- Modern and responsive UI  
- Smooth scrolling navigation  
- Hero section with introduction 
- About section with personal overview  
- Skills & tech stack   
- Projects showcase with links to live demos  
- Experience   
- Contact section  
- Fully deployed using GitHub Pages  

---

## 🧰 Tech Stack  
- **HTML5** – Markup & Structure  
- **CSS3** – Styling, Layout, Animations  
- **JavaScript** – Interactivity  
- **GitHub Pages** – Deployment

## 🚀 Run Locally  

### 1. Clone this repository  
```bash
git clone https://github.com/Kritika020704/Portfolio.git
```

### 2. Open the project folder  
```bash
cd Portfolio
```

### 3. Run the website  
Open the file:

```
index.html
```

Or use **Live Server** in VS Code for the best experience.
